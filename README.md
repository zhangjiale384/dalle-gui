## Size
![我的图片](./image/9.png "key")
你可以在这里选择图片的尺寸，DALL·E 2 和 DALL·E 3 所支持的尺寸有所不同，请你选择最合适的来作为你的生成模型：
- **DALL·E 2**： 
  - 1024x1024
  - 512x512
  - 256x256
- **DALL·E 3**：
  - 1024x1024
  - 1024x1792
  - 1792x1024

## Number of Images

## Revised Prompt
![我的图片](./image/19.png "key")
我们会使用 AI 对你的提示词进行修正，以便生成最符合你预期的图片。新生成的提示词会在 Revised Prompt 部分显示。
